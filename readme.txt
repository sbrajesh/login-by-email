== Description ==

Login by Email plugin gives the admins options to make their user login by username/email or force to login by email only.

= How it works:- =

Just activate the plugin. The default behaviour is to allow email login as optional. you can force users to use only email to login by checking the option in Dashboard->Settings->General

== Installation ==

This section describes how to install the plugin and get it working.


1. Download the zip file and extract
1. Upload `login-by-email` directory to the `/wp-content/plugins/` directory
1. Activate the plugin through the 'Plugins' menu.
1. Enjoy
